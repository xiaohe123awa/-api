# 蛋仔派对庄园api指定数据推送
## 项目介绍
通过蛋仔派对庄园数据api监测指定的数据并通过LLOneBot框架推送到QQ群
## 如何运行
1. 安装Python环境,本源码使用库均为标准库无需下载
2. 安装LLOneBot框架,并配置Http服务
3. 编辑main.py文件,修改其中的avatar_id为你的庄园id以及group_id,user_id为你的QQ群
4. 运行main.py文件
